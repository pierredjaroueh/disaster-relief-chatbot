# Disaster Relief Chatbot

A simple, web-based AI chatbot that provides disaster-relief guidance, emergency tips, and safety information.  
Built using HTML, CSS, and JavaScript with a clean and responsive interface.

---

## Features
- AI-style chat interface with smooth typing effect  
- Clean and modern UI  
- Mobile-responsive design  
- Automatic scroll-to-latest message  
- Easy to deploy with GitHub Pages  

---

## Tech Stack
- **HTML**
- **CSS**
- **JavaScript**

Deployment: **GitHub Pages**
